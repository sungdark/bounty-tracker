# Bounty Tracker - 赚钱机会跟踪库

## 说明
本仓库自动扫描并记录全网公开赚钱机会，包括：
- GitHub赏金任务
- Bug Bounty平台漏洞奖励
- 数字产品售卖机会
- 其他线上赚钱渠道

## 规则
- 法币优先，不设金额门槛
- 发现机会立即记录，同步更新进度
- 已完成任务标记结算状态
- 高价值任务重点标注优先级

## 目录结构
```
├── YYYY-MM-DD.md    # 每日扫描结果
├── completed/       # 已完成任务记录
└── README.md        # 说明文档
```
