# Bounty Tracker

自动扫描公开赚钱机会并同步记录，字段说明：
- `time`: 机会更新时间
- `url`: 机会链接
- `amount`: 赏金金额（法币优先）
- `assessment`: 初步评估结果
- `status`: 当前进度（待评估/进行中/已完成/已失效）
