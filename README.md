# Bounty Opportunity Tracker
自动扫描并记录所有赚钱机会，包括GitHub赏金、数字产品售卖、自由职业任务等。
## 字段说明
| 字段 | 类型 | 说明 |
|------|------|------|
| time | string | 发现时间 (UTC) |
| source | string | 来源平台 |
| title | string | 机会标题 |
| url | string | 详情链接 |
| amount | string | 金额/报酬 |
| status | string | 状态 (待评估/进行中/已完成/已过期) |
| notes | string | 备注 |
