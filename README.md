# Bounty Tracker
Track all money-making opportunities and bounty programs.
## Fields
- time: UTC timestamp when the opportunity was found
- type: Type of opportunity (github_bounty, digital_product, etc.)
- link: Direct link to the opportunity
- amount: Reward amount (currency + value)
- status: Status (discovered, in_progress, completed, paid, rejected)
- notes: Additional details
